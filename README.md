## Shortcuts

- `c`: Clear image
- `r`: Clear and randomize image
- `d`: Download image
- `o`: Open image
- `1-8`: Switch between color swatches
- `p`: Select Paint tool
- `f`: Select Fill tool
- `e`: Select Eraser tool
