This app uses the Hundred Rabbits Theme System. To change the theme, drag and drop a theme svg onto the window.

Shortcuts

      c: Clear image
      r: Clear and randomize image
      s: Save image
      o: Open image
      1-8: Switch between color swatches
      p: Select Paint tool
      f: Select Fill tool
      e: Select Eraser tool
